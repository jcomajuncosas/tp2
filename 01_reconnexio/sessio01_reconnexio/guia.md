## Objectius

- Reconnectar amb la sintaxi de Python (TP1) en un context sonor
- Entendre el so com a array de mostres (NumPy)
- Diferenciar el comportament de llistes vs. arrays (broadcasting)
- Generar un to pur i modular-ne freqüència i amplitud

## Connexions amb TP1

- `range()`/`for` (Loops & Lists) → reapareixerà al Bloc 4 (seqüenciació MIDI)
- Llistes `pattern = [...]` → arrays NumPy (canvi clau: operacions element a element)
- Funcions (`def ... return ...`) → funcions que retornen un array sencer (un so)
